# README
----

## Introduction
This project represent a node.js project that run by docker.

## Quick Start
 - Use make to quick start build and run node.js app with docker
```bash
$ make
```

- To stop
```bash
$ make stop
```

- Delete
```bash
$ make clean
```

- Other command
```bash
$ sh run.sh
```
